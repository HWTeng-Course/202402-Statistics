## Solution
Step1 \
$H_0$: $p_1$ = $p_2$ = $p_3$ = $p_4$ = $p_5$ \
$H_a$ : not all $p$ are equal

Step2 \
Set $\alpha$ = 0.05

Step3 \
test statistics: $\chi^2$ = $\sum_{i=1}^{a}$ $\sum_{i=1}^{b}$ $\frac{\displaystyle (O_i-E_i)^2}{E_i}\$  $\approx x^2_{(a-1)(b-1)}$

Step4 \
$\chi^2_0$ = $\frac{{(73-65)^2}}{65}+ \frac{{(57-65)^2}}{65}+ \frac{{(12-13.5)^2}}{13.5}+ \frac{{(15-13.5)^2}}{13.5}+ \frac{{(6-8.5)^2}}{6}+ \frac{{(11-8.5)^2}}{8.5}+ \frac{{(4-6.5)^2}}{6.5}+ \frac{{(9-6.5)^2}}{6.5}+ \frac{{(3-4.5)^2}}{4.5}+ \frac{{(6-4.5)^2}}{4.5}=6.831975$ with (a-1)(b-1) = 1*4 = 4df$\

Step5 \
The rejection region with $\alpha$ = 0.05 and 4 $\df$ is $\chi^2$ > 0.7107

Step6 \
$\H_0$ is not rejected, and the results are declared non-significant. There is insufficient evidence to indicate a difference in spending patterns between males and females.
